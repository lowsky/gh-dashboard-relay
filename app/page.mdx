## Purpose

It shows GitHub Feature Branches together with their build state and link to each deployment, for live view of each branch.

Start here for [loading a user's repos](./relay) and dive into example repo based [on relay](https://relay.dev/)

Start here for [loading a user's repos](./apollo) and dive into example repo based [on apollo](https://www.apollographql.com/)

### Auth\*

After accepting this web app to authenticate against GitHub, it will fetch information from
GitHub via GraphQL.

All will be done and cached only in the browser on the client side.

It keeps the generated auth token for it in the cookie üç™.
You will be able to logout to delete it completely.

## Tech stack

- [NEXTJS](https://nextjs.org/) - app router
- [Relay](https://relay.dev/)
- [GraphQL](https://graphql.org/)
- [GitHub GraphQL api](https://graphql.github.com)
